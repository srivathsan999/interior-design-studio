# Atelier Interior - Premium Interior Design Studio Template

A premium, production-ready HTML template for interior design studios. Built with TailwindCSS, featuring light/dark mode, glassmorphism effects, and fully responsive design.

## Features

- **13 Complete Pages**: Home (2 variants), About, Services, Projects, Blog, Pricing, Contact, Login, Signup, and 404
- **Light/Dark Mode**: Seamless theme switching with CSS variables
- **Premium Design System**: Carefully crafted color palette, typography, and spacing
- **Glassmorphism & Neomorphism**: Modern UI effects applied appropriately
- **Fully Responsive**: Mobile-first design with breakpoints for all devices
- **WCAG AA Compliant**: Accessible with proper contrast ratios
- **Smooth Animations**: Elegant page transitions and hover effects
- **Interior Design Focus**: All imagery and content tailored for interior design studios

## Design System

### Color Palette

**Light Mode:**
- Background: #F6F5F3
- Surface: rgba(255,255,255,0.7)
- Primary Text: #1E1E1E
- Secondary Text: #5F5F5F
- Accent: #C9A46A (Luxury gold)
- Borders: rgba(0,0,0,0.08)

**Dark Mode:**
- Background: #0E0E11
- Surface: rgba(255,255,255,0.08)
- Primary Text: #F2F2F2
- Secondary Text: #B5B5B5
- Accent: #D6B676
- Borders: rgba(255,255,255,0.12)

### Typography

- **Headings**: Playfair Display
- **Body/UI**: Inter
- **Buttons/Nav**: Inter Medium
- **Line Height**: Body 1.6, Headings 1.25

### Border Radius

- Cards: 16px
- Buttons: 12px
- Inputs: 10px

## Usage

1. Open any HTML file in a modern web browser
2. The theme toggle is available in the navbar
3. All pages are fully functional with navigation
4. Forms are UI-only (no backend integration)

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

All design tokens are defined in `assets/css/theme.css` using CSS variables. Modify these variables to customize colors, fonts, and spacing throughout the template.
